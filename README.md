# eKayaLink

eKayaLink is a blockchain-powered remittance platform designed to connect Overseas Filipino Workers (OFWs) with their families in the Philippines through fast, affordable, and secure money transfers.

## Team

**eKaya Innovators**

## Hackathon

APAC Stellar Hackathon 2026

## Problem

Millions of OFWs send money home regularly but often face high fees, slow processing times, and limited accessibility.

## Solution

eKayaLink leverages the Stellar network to facilitate fast and low-cost remittances while integrating with PDAX for Philippine Peso (PHP) cash-in and cash-out services.

## Planned Features

- User Authentication
- Digital Wallet Dashboard
- Send Money
- Transaction History
- Stellar Testnet Integration
- PHP Cash-In
- PHP Cash-Out
- Real-Time Exchange Rate Display
- Transaction Tracking
